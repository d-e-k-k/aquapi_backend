# AquaPi Backend

## Description
AquaPi is an automated aquarium controller that can monitor and manage a variety of aquarium components. ## Instillation Steps
1. Fork and clone this repo
2. Cd into the project directory
3. Install project dependencies by running npm i 
4. Create a new feature branch and start working!
